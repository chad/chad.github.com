One Million Gems Served

Sep 18th, 2005

As Jim put
so eloquently,

        $ lynx -dump http://gems.rubyforge.org/stats.html \
         | ruby -e 'p ARGF.inject(0) { |s,t| t.split[1].to_i + s }'
        $ 1007130

Also note that there are now 396 unique libraries and applications to
choose from in the repository. 100 new gems in only 3
months! Got a new library that you’ve been planning to polish off
and release? Go ahead and spend your Sunday on it and you might be lucky
number 400.

Perhaps it’s not too late to see RubyGems make an appearance in Ruby
1.8.3?
