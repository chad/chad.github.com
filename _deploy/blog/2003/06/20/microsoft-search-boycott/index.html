Microsoft Search Boycott

2003.06.20

Apparently, Microsoft is brewing up their own new search engine technology,
which has got some people thinking they’re about to try to
Netscape-ize Google (scriptingnews.userland.com/stories/storyReader$2071).
I don’t really buy into the idea that any company is evil (search.msn.com/results.aspx?q=%22microsoft+is+evil%22&FORM=SMCRT),
but I still like this idea: www.idlewords.com/boycott.pl.

The idea is that, if enough people setup their robots.txt to disallow
Microsoft’s bot, it will create holes in MS’s relevance
ranking, rendering the search engine ineffective. I don’t want to
ruin Microsoft. But, it would be extremely interesting if a simple,
distributed idea like this actually had an effect on a seemingly inevitable
outcome.

So, what are you waiting for? Join in! The numbers are small now.

The disadvantage here is that when Microsoft does kill the
competition, your site won’t be indexed. I think can live with that
for at least as long as it takes to `rm robots.txt` and get crawled.
